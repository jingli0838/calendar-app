# Calendar Backend

This is the backend server for the Calendar App, built with **TypeScript** and **Express**.

## Tech Stack

- **TypeScript**
- **Express.js**
- **Zod** (for data validation)
- **CORS** (for cross-origin requests)
- **Nodemon** + **ts-node** (for development)

---

## Project Setup

### Install dependencies

```bash
npm install
```
